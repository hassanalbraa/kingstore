# **App Name**: KING STORE

## Core Features:

- User Authentication: Secure login and registration system for users and admins.
- User Dashboard: Personalized dashboard displaying user's name, balance, and available game offers.
- Game Offer Display: Display game offers as interactive cards, highlighting price and details.
- Admin Interface: Admin panel for managing users, balances, and offer prices.
- User Profile Management: Allow users to change their passwords via a settings page.
- RTL Support: Ensure the entire application supports right-to-left text direction for Arabic language users.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a calm, inviting gaming atmosphere.
- Background color: Light off-white (#F0F4FF) to provide a clean, unobtrusive backdrop.
- Accent color: Darker blue (#1E88E5) for interactive elements like buttons and hover states, providing clear affordance.
- Body and headline font: 'Poppins' for a clear, modern aesthetic. Note: currently only Google Fonts are supported.
- Central content area with rounded corners and soft shadows for a modern look, optimized for both mobile and desktop.
- Subtle hover effects on buttons and offer cards to provide interactive feedback.
